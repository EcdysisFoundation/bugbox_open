How It Works
  1. Enter the cluster number (e.g., C1, C2)
  2. Enter the year
  3. System finds the latest label generation for that cluster
  4. Extracts all transect codes from the label generation
  5. Creates 5 rows per transect (one for each depth: 0-5, 5-10, 10-15, 15-30, 30-60 cm)
  6. Populates grower names from submitted applications
  7. Downloads Excel file ready for submission
