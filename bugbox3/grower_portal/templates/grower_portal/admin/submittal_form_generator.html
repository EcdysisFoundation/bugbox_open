How It Works
  1. Enter the cluster number (e.g., C1, C2)
  2. Enter the year
  3. Select which submittal forms to generate (Soil and/or Plant)
  4. System finds the latest label generation for that cluster
  5. For Soil: Creates 5 rows per transect (one for each depth: 0-5, 5-10, 10-15, 15-30, 30-60 cm)
  6. For Plant: Creates 1 row per transect with forage type (only for transects with forage labels)
  7. Populates grower names from submitted applications
  8. Downloads Excel file(s) ready for submission
