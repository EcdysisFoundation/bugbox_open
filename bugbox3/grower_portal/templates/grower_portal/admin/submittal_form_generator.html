How It Works
  1. Select project type (Avalanche or Ignite)
  2. Enter cluster number and year
  3. Tab out of the year field to load inner label batches for that project
  4. Pick the label generation to use (if several exist)
  5. Check which codes to include on the submittal form(s)
  6. Choose Soil and/or Plant submittal forms
  7. For Soil: 5 rows per selected code (depths 0-5, 5-10, 10-15, 15-30, 30-60 cm)
  8. For Plant: 1 row per selected code when the batch includes forage
  9. Grower names are filled from submitted applications when codes match
